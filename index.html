About BoskaCoin

BoskaCoin is a cryptocurrency forked from LebowskisCoin. It uses the Scrypt algorithm and the block reward is 20 BOSKA per block as a base reward. But, the reward can be more depending on:

  • Luck, it is random whether it is 20, 200, 500, 2500 or 10000 BOSKA. (The higher the reward, the lower the chance of getting it)
  • Transaction fees, if there are transaction fees in the block, they will be added to the block reward.

Blocks are generated about every minute when the network is active.

Wallet

You can download the BoskaCoin wallet from our GitHub releases page. The wallet is available for Windows, and Linux binaries can be compiled from source.

Make sure to always download the wallet from github.com/boskacoin/boskacoin to ensure you have the official version.

Connect to the Network

To connect your wallet to the BoskaCoin network, you can use the following seed nodes:

  • pyrrhocorisapterus.org:8085 - Seed node maintained by Imusing
  • cavesystem.net:8085 - Seed node maintained by Novixx
  • seed.boskacoin.org:8085 - Seed hosted by BoskaCoin.org
