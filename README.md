# boskacoin-website

This repository contains the source code for the Boskacoin official website.

Images are public domain, including the BoskaCoin logo.

## Contributing

We are looking for someone who can make the website look better, because currently it looks very basic.
If you want to contribute, please fork the repository and submit a pull request with your changes.

If no one can do it, leaving it like this is also OK, might just be the style of the BOSKA coin (:
